# A股智投 (AI Stock Analyst) 使用说明

这是一款基于 Google Gemini 3 Flash 模型开发的专业 A 股分析工具，旨在通过 AI 搜索增强技术为投资者提供最新的市场洞察。

## 🚀 核心功能
1. **实时行情解读**：自动搜索并解析目标股票的最新股价、涨跌幅及成交量。
2. **深度技术面分析**：基于最新 K 线形态提供短期走势判断。
3. **基本面与消息面**：汇聚近期新闻、研报精华，分析主力资金动向。
4. **操作建议**：提供明确的短线及中线投资逻辑参考。

## 📱 手机端使用
- **直接搜索**：在搜索框输入股票名称（如“贵州茅台”）或代码（如“600519”）。
- **点击建议**：首页提供了热门板块建议，点击即可一键分析。
- **查看来源**：分析结果下方会列出 AI 引用的网页链接，点击可跳转查看原始资讯。

## 💻 电脑端使用
- **宽屏体验**：自动适配大屏幕，图表更清晰。
- **快速复制**：方便将 AI 生成的分析报告复制到文档或交易软件。

## ⚠️ 重要声明
- **风险提示**：本工具由 AI 生成，数据来源于公开搜索，不构成任何法律约束力的投资建议。
- **配色习惯**：应用遵循中国股市习惯，**红色代表上涨**，**绿色代表下跌**。
- **网络说明**：本应用在支持 Gemini API 的环境下可正常使用。
